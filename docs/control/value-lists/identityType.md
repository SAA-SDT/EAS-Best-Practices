# identityType
**Scope Note:**
Indicates whether the identity is given or acquired. Recommended that it be used when multiple identities are described in the same EAS instance using `<multipleIdentities>`. Enabled processors to distinguish between the description of a person and one or more personae.

- ISAAR(CPF), Glossary of terms and definitions, [https://www.ica.org/app/uploads/2023/12/CBPS_Guidelines_ISAAR_Second-edition_EN.pdf](https://www.ica.org/app/uploads/2023/12/CBPS_Guidelines_ISAAR_Second-edition_EN.pdf)
- Records in Contexts Conceptual Model version 1.0, RiC-E07, [https://www.ica.org/app/uploads/2024/01/ric-cm-1.0_0.pdf](https://www.ica.org/app/uploads/2024/01/ric-cm-1.0_0.pdf)

## acquired
**Scope Note:**
Used to indicate a name by which an entity is known that differs from a legal name or birth name. An acquired name may or may not be the authorized version. In the case of a group pseudonym, for example, there may be an authorized version of the pseudonym as well as the names of each individual.

## given
**Scope Note:**
Used to indicate a legal or birth name. May or may not be the authorized version.
