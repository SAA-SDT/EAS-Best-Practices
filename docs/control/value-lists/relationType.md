# relationType
**Scope Note:**
To define the type of relation between the entity described in the EAS instance and a related entity.

---

## authority
**Scope Note:**
Any control, ownership, or management relation between agents and other types of entities.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#hasOrHadAuthorityOver](https://www.ica.org/standards/RiC/RiC-O_1-1.html#hasOrHadAuthorityOver)

## familialOrganizational
**Scope Note:**
Family or organizational relation between agents, which is not necessarily defined or characterized as hierarchical or temporal (also including generic work or membership relations).

- Inspired by values used in EAC-CPF 2010 (revised 2018)

## hierarchical
**Scope Note:**
Any subdivision relation between the entity being described in the EAS instance and the related entity; this might be the relation between subordinate and superior agents, the isPartOf/hasPart relation between record resources, etc.

- Inspired by values used in EAC-CPF 2010 (revised 2018) and terminology used in RiC-O

## identity
**Scope Note:**
Relation between to entities that actually are the same thing.

- Inspired by values used in EAC-CPF 2010 (revised 2018) and terminology used in [https://www.w3.org/TR/owl-ref/#sameAs-def](https://www.w3.org/TR/owl-ref/#sameAs-def)

## performance
**Scope Note:**
Relation between agents and functions.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#PerformanceRelation](https://www.ica.org/standards/RiC/RiC-O_1-1.html#PerformanceRelation)

## provenance
**Scope Note:**
Relation between agents and functions on the one hand and record resources and instantiations on the other hand that they have created and/or accummulated or resulted in.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#hasOrganicOrFunctionalProvenance](https://www.ica.org/standards/RiC/RiC-O_1-1.html#hasOrganicOrFunctionalProvenance)

## related
**Scope Note:**
Most generic relation that connects any entity with any other entity; should only be used if none of the more specific relations can be applied.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#isRelatedTo](https://www.ica.org/standards/RiC/RiC-O_1-1.html#isRelatedTo)

## sequential
**Scope Note:**
Arrangement relation indicating that one entity follows - or is followed by - another, which is not necessarily defined or characterized chronologically, e.g. the order in which records are arranged in a record set.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#precededInSequence](https://www.ica.org/standards/RiC/RiC-O_1-1.html#precededInSequence)

## spatial
**Scope Note:**
Spatial relation between the entity being described in the EAS instance and a place to which the existence and lifecycle of the entity being described is associated.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#isPlaceAssociatedWith](https://www.ica.org/standards/RiC/RiC-O_1-1.html#isPlaceAssociatedWith) and [https://www.ica.org/standards/RiC/RiC-O_1-1.html#isOrWasLocationOf](https://www.ica.org/standards/RiC/RiC-O_1-1.html#isOrWasLocationOf)

## subject
**Scope Note:**
Relation between record resources and other entities that are the (main) subject(s) of the resource.

- Adapted from [https://www.ica.org/standards/RiC/RiC-O_1-1.html#hasOrHadSubject](https://www.ica.org/standards/RiC/RiC-O_1-1.html#hasOrHadSubject)

## temporal
**Scope Note:**
Any predecessor/successor relation between agents and functions, but also e.g. original/copy relation between record resources and  instantiations derived from each other etc.

- Inspired by values used in EAC-CPF 2010 (revised 2018) and terminology used in RiC-O
