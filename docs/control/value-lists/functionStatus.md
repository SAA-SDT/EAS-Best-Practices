# functionStatus
**Scope Note:**
For detailing the current status of the function being described.

- EAF 1.0 Tag Library, used in attribute functionStatus when EASlist is the source of values. Inspired by maintanceStatus and maintenanceEventType

---

## active
**Scope Note:**
Indication that the described function is currently in active use.

## deleted
**Scope Note:**
Indication that the described function has been deleted and is no longer in use.

## draft
**Scope Note:**
Indication that the description of the function is in progress.

## final
**Scope Note:**
Indication that the description of the function has been finalised.

## inactive
**Scope Note:**
Indication that the described function is not currently in active use.

## obsolete
**Scope Note:**
Indication that the described function is not currently used but is retained for reference.

## revised
**Scope Note:**
Indication that any type of general modification to the described function has been made.

## unknown
**Scope Note:**
Indication that the status of the described function is not known.
